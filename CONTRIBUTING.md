# Contributing

## Get started
To make any code changes, you first need to fork this repo and clone it on your local machine. Note: your machine MUST be a mac. Only Macs can run Xcode.

Next, you must [install Xcode](https://developer.apple.com/xcode/), and sign in using your Apple ID. 
Due to Apple capability restrictions, unless you have a paid developer account, some features will not work on your local clone, such as WeatherKit, or push notifications.

After that, you are able to start developing. Before you build the product to your phone or a simulator, you must configure the signing, 
which can be found in the Signing & Capabilities section on the main app target. 

Finally, you can build, test, and when OK, you can make a PR and make your changes live!
